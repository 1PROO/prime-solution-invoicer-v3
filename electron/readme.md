دليل إنشاء ملف EXE (التطبيق المكتبي)
لقد تم تجهيز المشروع الآن ليعمل كتطبيق سطح مكتب (Windows EXE). اتبع الخطوات التالية لإنشاء الملف القابل للتوزيع لأول مرة ولعمل تحديثات مستقبلاً.

1. تثبيت أدوات البناء
أول مرة فقط، تحتاج لتثبيت المكاتب اللازمة. افتح التيرمينال داخل مجلد المشروع ونفذ:

npm install electron electron-builder wait-on concurrently cross-env --save-dev
2. تجربة التطبيق محلياً
للتأكد من أن كل شيء يعمل قبل البناء، يمكنك تشغيل نسخة المطور:

npm run electron:dev
سيفتح هذا التطبيق في نافذة مستقلة بدلاً من المتصفح.

3. بناء ملف EXE
لإنشاء الملف النهائي الذي سترسله لصديقك:

npm run electron:build
هذه العملية قد تأخذ بضع دقائق. بعد الانتهاء، ستجد ملف التثبيت في مجلد: release/Prime Invoicer Setup 0.0.0.exe

طريقة إرسال التحديثات
بما أن صديقك "لا علاقة له بالبرمجة"، فإن أسهل طريقة للتحديث هي:

قم بعمل التعديلات التي تريدها على الكود في جهازك.
قم بتشغيل الأمر npm run electron:build مرة أخرى. نصيحة: يفضل تغيير رقم الإصدار في ملف 
package.json
 (مثلاً من 0.0.0 إلى 0.0.1) قبل البناء ليعرف النظام أنه تحديث.
أرسل ملف exe الجديد لصديقك.
عندما يثبته صديقك، سيقوم بتحديث النسخة القديمة تلقائياً مع الحفاظ على بياناته المحفوظة (لأن البيانات تخزن في مجلد المستخدم في الويندوز وليس في مجلد البرنامج).
الميزات الجديدة التي تم إضافتها
PDF ذكي: الفواتير الطويلة الآن تضغط نفسها لتناسب صفحة A5 واحدة تلقائياً.
عروض الأسعار: يمكنك اختيار "نوع المستند: عرض سعر" من الإعدادات، وسيتحول القالب لنمط عرض السعر (تاريخ العرض، صالح حتى...).
حفظ العملاء: يمكنك الآن حفظ بيانات العميل (اسم، عنوان، هاتف) واسترجاعها بضغطة زر عند كتابة الاسم، مع ميزة الإكمال التلقائي.
